# Angular-Demo
# Local Event Board
This is a hands-on learning project for practicing modern Angular. Use it to explore and experiment with common Angular patterns and APIs.

Topics covered
---------------
- Standalone components and component composition.
- Routing and navigation
- lazy-loading, dynamic `events/:id`, wildcard route
- Signals for local reactive state (`signal`) and shared UI state via services
- Dependency injection
- Structural directives (`NgIf`, `NgFor`)
- Reactive forms

See the app source under the `src/app` folder to explore examples of each topic.
